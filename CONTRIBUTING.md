# Guia de Contribuição - Projeto Gênesis

Para contribuir, você deve ser capaz de operar o terminal Linux/macOS e entender o pipeline de física em `core/physics.py`.

### Regras de Ouro:
1. Nunca suba arquivos `.parquet` para o repositório.
2. Se o Win Rate cair abaixo de 55%, o Pull Request será rejeitado.
3. Documente cada recalibragem de sensibilidade.
